10.2 — Systems of Three Linear Equations

Methods and ideas from the previous section extend naturally to three-dimensional systems. We will introduce some new ideas towards matrix representations.

General form of a three-dimensional linear system

\begin{align*} a_{11}x + a_{12}y + a_{13}z &= b_1 \\ a_{21}x + a_{22}y + a_{23}z &= b_2 \\ a_{31}x + a_{32}y + a_{33}z &= b_3 \end{align*}

Some extra terminology

We will call the equations in a linear system rows and abbreviate the $i$th row by $r_i$. For example, equation 2 in the above would be called $r_2$.

Row assignment: suppose $r$ is some equation. The notation$$r_i \leftarrow r$$means replace row $i$ with whatever $r$ is.

Row Operations

We can perform certain operations on a linear system that will not change the solution type or the solution itself if it is unique. The following are valid row operations:
  1. $r_i \leftrightarrow r_j$ (swap row $i$ with row $j$).
  2. $r_i \leftarrow \alpha r_i$ (multiply row $i$ by some nonzero constant, $\alpha$ and put the result in row $i$)
  3. $r_i \leftarrow \alpha r_i + \beta r_j$ (multiply row $i$ by some nonconstant, $\alpha$ and add a constant multiple, $\beta$, of row $j$ and put the result in row $i$)
Use row operations to describe the solution type of the following system: \begin{align*} x + y + z &= 1 \\ 2x + 2y + 2z &= 2 \\ 3x + 3y + 3z &= 3 \end{align*}
Infinitely many solutions.
Use row operations to describe the solution type of the following system: \begin{align*} x + y + z &= 1 \\ 2x + 2y + 2z &= 2 \\ 3x + 3y + 3z &= 6 \end{align*}
No Solution.
Use row operations to describe the solution type of the following system: \begin{align*} 3x + 5y - z &= -7 \\ x + y + z &= -1 \\ x + 3y - 3z &= 5 \end{align*}
No solution.

Recognizing solution types

  1. Inconsistent equations always means there is no solution.
  2. Consistent but redundant equations always mean infinite solutions (only when there are no inconsistent equations present).
On the other hand, if row operations yield a system like the following: \begin{align*} a_{11}x + a_{12}y + a_{13}z &= b_1 \tag{1}\\ a_{22}y + a_{23}z &= b_2 \\ a_{33}z &= b_2 \end{align*} and $a_{11} \neq 0, a_{22} \neq 0, a_{33} \neq 0$, then the system has a unique solution because we can solve for $z$ easily in $r_3$, substitute $z$ into $r_2$ to find $y$, and finally substitute $y$ and $z$ into $r_1$ to solve for $x$. This process is called back substitution.

More Terminology

If row operations turn a system into one like $(1)$ labelled above, then it is said to be in upper triangular form with $a_{11}, a_{22}, a_{33}$ being the diagonal elements.

Tip: if we're only interested in the type of solution, then back-substitution is not needed. We only need to have a system in upper triangular form with non-zero diagonal elements to have a unique solution. If we want to know what the unique solution is, then we must perform back-substitution to find it.
Use row operations to describe the solution type of the following system: \begin{align*} x - y - z &= -4 \\ 2x - 3y + 4z &= -15 \\ 4x - 5y + 6z &= -23 \end{align*}
Unique Solution.
Use row operations to describe the solution type of the following system: \begin{align*} 2x + 4y - 10z &= -2 \\ 3x + 9y - 21z &= 0 \\ x + 5y - 12z &= 1 \end{align*}
Unique Solution.
