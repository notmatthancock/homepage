4.4 — Properties of Logarithms

The following can be derived using the logarithm definition:
  1. $\log_a(xy) = \log_ax + \log_ay$
  2. $\log_a(\frac{x}{y}) = \log_ax - \log_ay$
  3. $\log_a(x^p) = p\log_ax$
Solve for $y$: $\;\; \ln y = x - \ln 2 $
$\frac{e^x}{2}$
Simplify the expression: $\;\;e^{2\ln 4 + \ln 3}$
$48$

One-to-one Property

Both $f(x) = a^x$ and $f(x) = \log_a(x)$. This can easily seen by recalling the graphs and using the horizontal line test.

Hence,
  1. $a^x = a^y \Rightarrow x = y$
  2. $\log_ax = \log_ay \Rightarrow x = y$

Solve for $y$: $\;\;\ln y = \ln(x-4)$
$y=x-4$

Expanding Logarithmic Expressions

We can use the addition, subtraction, and power identities to expand single log expressions into an expression involving many logs.
Expand as much as possible: $\;\;\displaystyle \log \left( \frac{x^2\sqrt[3]{y-1}}{w^2z^5} \right)$
$2\log x + \frac{1}{3}\log(y-1) - 2\log w - 5\log z$
Expand as much as possible: $\;\;\displaystyle \ln \left( \frac{\sqrt{x}}{y^4(z-3)^{-2}} \right)$
$\frac{1}{2} \ln x + 2\ln(z-3) - 4\ln y$
If $\log_63 = a$ and $\log_64 = b$, then $\log_6\frac{16}{3} =$ ?
$4b - a$

Collapsing Multiple Logarithmic Expressions

We can also use the addition, subtraction, and power identities in reverse order to collapse many logs into one (assuming they are the same base).
$\frac{1}{3}\log z - 3 \log x + 5 \log y =$ ?
$\displaystyle \log \left( \frac{\sqrt[3]{z}\,y^5}{x^3} \right)$

Change of Base Formula

All logarithms are equivalent up to a scaling factor: $$ \log_ax = \left( \frac{1}{\log_ba} \right) \log_bx = \frac{\log_bx}{\log_ba} $$ For instance (changing from base 4 to base 2): $$ \log_4x = \frac{\log_2x}{\log_24} = \frac{1}{2} \log_2x $$
Which of the following statements are true if $x \gt 0$, $y \gt 0$, $x \neq 1$, and $y \neq 1$?
