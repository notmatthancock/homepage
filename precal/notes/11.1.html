11.1 — Sequences

Consider the list:
$$ 1, 3, 5, \ldots $$ This list is called sequence. More specifically, a sequence is function whose domain is the integers. So for the above, $$ f(1) = 1, f(2) = 3, f(3) = 5, \ldots $$ We call each item in the sequence a term. We usually denote terms in a sequence with a subscript. The $n$th term in above sequence would be: $$ f(n) = a_n, \; n = 1, 2, 3, \ldots $$ Often the whole sequence is denoted as: $\{a_n\}$. In the above case, we can explicitly define the $n$th term in the sequence: $$ a_n = 2n-1 $$
Write the first three terms from the sequence, $\displaystyle \left\{ \frac{n}{n+2} \right\}$.
$\displaystyle \frac{1}{3}, \frac{2}{4}, \frac{3}{5}$
The sequence in the next example is called an alternating sequence.
Write the first four terms from the sequence, $\displaystyle \left\{ \left( \frac{-1}{2} \right)^{n+1} \right\}$.
$\displaystyle \frac{1}{4}, \frac{-1}{8}, \frac{1}{16}, \frac{-1}{32}$
Find the $n$th term for the sequence: $$ \frac{1}{3}, \frac{1}{9}, \frac{1}{27}, \ldots $$
$\displaystyle a_n=\left(\frac{1}{3}\right)^{n-1}$

Recursive Sequences

A sequence whose $n$th term is defined in terms of the previous terms, is called a recursive sequence. For such a sequence, we need initial term(s) to begin writing out the sequence.
Let $a_1 = 1$, $a_2 = 1$, and $a_n = a_{n-1} + a_{n-2}$ for $n \geq 3$. Write the first 7 terms of this sequence.
$1, 1, 2, 3, 5, 8, 13$
The following example is called the factorial function and is denoted: $a_n = n!$.
This sequence starts at 0. Let $a_0 = 0! = 1$ and $a_n = n! = n \cdot a_{n-1} = n \cdot (n-1)!$ for $n \geq 1$.

Write out the first five terms of this sequence.
$1, 1, 2, 6, 24$

Series

A series is a sum of a sequence. It might be finite or infinite. For now, we will consider finite series.

Recall summation notation. The sum of the first $N$ terms in a sequence, $\{ a_n \}$, would be denoted: $$ \sum_{n=1}^N a_n = a_1 + a_2 + \cdots + a_N $$ Some properties of sums:
  1. If $K$ is a constant, then $$\sum_{n=1}^N K a_n = K \sum_{n=1}^N a_n$$
  2. If $\{a_n\}$ and $\{ b_n \}$ are two sequences, then $$\sum_{n=1}^N (a_n + b_n) = \sum_{n=1}^N a_n + \sum_{n=1}^N b_n$$
Compute the sum for $$ \sum_{j=2}^4 j^2 - 2j $$
$13$
Write out the sum for $$ \sum_{i=1}^{100} f(x_i) \Delta x $$ if $f(x_i) = 2i$ and $\Delta x = 0.1$.
$0.2 + 0.4 + 0.6 + \ldots + 20$
Write out the sum for $$ \sum_{n=1}^5 (-1)^{n-1} \ln\left(\frac{n+2}{2n-1}\right) $$
$\displaystyle \ln\left(\frac{49}{24}\right)$
