Exponent rules

  1. $a^0 = 0$
  2. If $n$ is an integer, $\sqrt[n]{a} = a^{1/n}$.
  3. $a^xa^y = a^{x+y}$
  4. $\frac{a^x}{a^y} = $$a^xa^{-y} = a^{x-y}$
  5. $a^xb^x = (ab)^x$
  6. $(a^x)^y = (a^y)^x = a^{xy}$

Logarithm definition

For $a \gt 0$, $$ a^x = y \Leftrightarrow x = \log_a(y) $$ $e = 2.7182\ldots$

$\ln(x) := \log_e(x)$ is called the natural log.

Logarithm rules

  1. $\log_a(a) = 1$
  2. $\log_a(1) = 0$
  3. $\log_a(x^y) = y \log_a(x)$
  4. $\log_a(xy) = \log_a(x) + \log_a(y)$
  5. $\log_a(x/y) = \log_a(x) - \log_a(y)$
